Вариативный макрос для вывода форматированных сообщений об ошибках.

Направляет текстовый поток напрямую в стандартный вывод ошибок (stderr). Избавляет от необходимости вручную прописывать fprintf(stderr, ...) в каждом блоке.

Parameters
formatСтрока форматирования в стиле printf (содержит спецификаторы типа %s, %d и т.д.).
...Переменное число аргументов, подставляемых в строку форматирования.
Note
Требует обязательного наличия хотя бы одного аргумента для замены __VA_ARGS__ в стандарте C99.

Главная точка входа в программу.

Функция выполняет первичную инициализацию, проверяет количество переданных аргументов и валидирует ключевой флаг -n. Если проверки не пройдены, функция активирует макрос LOG_ERROR и завершает работу операционной системы с кодом ошибки.

Parameters
[in]argcКоличество аргументов командной строки, переданных при запуске.
[in]argvМассив указателей на строки (сами аргументы).
  • argv[0]: Путь к исполняемому файлу.
  • argv[1]: Обязательный параметр управления (ожидается "-n").
  • argv[2]: Дополнительный пользовательский параметр.
Returns
Статус завершения программы.
Return values
0Успешное выполнение программы, все параметры прошли валидацию.
1Критическая ошибка: недостаточно аргументов или передан неверный параметр.
Attention
Программа немедленно прекращает выполнение через вызов exit(1), если нарушены правила входных данных.
