Макрос автоматической генерации и инициализации командных строк.

Этот макрос инкапсулирует рутинный процесс создания строковых переменных для сверки команд. Он выполняет три важные задачи за один шаг:

  1. Выделяет буфер размером 256 байт в стеке.
  2. Формирует уникальное имя переменной путем конкатенации суффикса _cmd (оператор ##).
  3. Безопасно копирует строковое представление имени макроса (оператор #) в созданный буфер.
Parameters
NAMEИдентификатор команды в виде чистого текста (например, copy, paste).

int execute_command_interpreter ( void  )

Обработчик и интерпретатор текстовых команд пользователя.

Функция считывает из стандартного потока ввода (stdin) имя команды, инициализирует локальный пул доступных инструкций (copy, paste, cut) с помощью специализированного макроса CMD, а затем выполняет каскадное сравнение строк для определения выбранного действия.

Note
Функция ожидает на входе строку без пробельных символов. При вводе строки, превышающей размер буфера, возможно переполнение.

Схема работы функции:

  1. Инициализация эталонных строк команд (copy_cmd, paste_cmd, cut_cmd).
  2. Ожидание ввода от пользователя через scanf.
  3. Поочередное сравнение через strcmp.
  4. Завершение работы с возвратом статуса.
Returns
Порядковый статус завершения функции.
Return values
0Успешное выполнение, разбор команд завершен штатно.

Разворачивается в: char copy_cmd[256] = ""; strcpy(copy_cmd, "copy"); Создает эталон для команды копирования.

Разворачивается в: char paste_cmd[256] = ""; strcpy(paste_cmd, "paste"); Создает эталон для команды вставки.

Разворачивается в: char cut_cmd[256] = ""; strcpy(cut_cmd, "cut"); Создает эталон для команды вырезания.

Локальный буфер для хранения сырого ввода пользователя. Выделено 256 байт, что совпадает с размером эталонных строк команд.

Блокирующее чтение строки из консоли. Символ s считывает поток символов до первого пробела, табуляции или переноса строки.

Проверка на совпадение с командой "copy". Сравнение возвращает 0, если строки полностью идентичны.

Проверка на совпадение с командой "paste". Вызывается, если предыдущее условие не сработало или для независимого анализа.

Проверка на совпадение с командой "cut". Финальная ветка валидации известного пула команд.
