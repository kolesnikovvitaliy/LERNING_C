# 🛠️ Variadic Macro Demo & Argument Validator
Демонстрационный проект на языке СИ, иллюстрирующий механизмы развертывания **вариативных макросов** (`variadic macros`) и построение строгой системы валидации аргументов командной строки.

---

## 📌 Обзор проекта

Данное приложение выполняет первичную инициализацию и жесткую проверку параметров запуска. Главная особенность — использование кастомного макроса логирования `#LOG_ERROR`, который перенаправляет форматированный поток данных в системный тракт ошибок `stderr`, автоматически подставляя метаданные текущей сборки.

### Ключевой функционал
* **Вариативное логирование**: Использование спецификатора `__VA_ARGS__` для динамического форматирования строк.
* **Потоковая изоляция**: Разделение стандартного вывода (`stdout`) и ошибок (`stderr`).
* **Параметрический контроль**: Двухуровневая валидация флагов конфигурации перед запуском основного бизнес-процесса.

## 💡 Глубокий разбор: Вариативные макросы

В основе логирования лежит препроцессорная конструкция:
```c
#define LOG_ERROR(format, ...) \
    fprintf(stderr, format, __VA_ARGS__)
```

### Как это работает под капотом:
1. **`...` (Эллипсис)**: Указывает препроцессору, что макрос принимает переменное число аргументов.
2. **`__VA_ARGS__`**: Специальный идентификатор, вместо которого компилятор подставляет все аргументы, переданные на место троеточия.
3. **Развертывание**: Вызов `LOG_ERROR("Error %d: %s", code, msg)` превращается в явный `fprintf(stderr, "Error %d: %s", code, msg)`.

> ⚠️ **Важное замечание:** Согласно стандарту C99, макрос ожидает как минимум один аргумент для замены `__VA_ARGS__` (помимо строки формата).

## 📋 Спецификация аргументов командной строки

Программа ожидает строго определенную структуру параметров:

| Индекс | Параметр | Тип | Описание | Обязательность |
| :--- | :--- | :--- | :--- | :--- |
| `argv[0]` | `path_to_binary` | Строка | Путь к исполняемому файлу программы | Автоматически |
| `argv[1]` | `-n` | Строка | Ключевой управляющий флаг инициализации | **Да** |
| `argv[2]` | `value` | Строка | Пользовательские данные / Дополнительный параметр | **Да** |

## 🚀 Сборка и запуск

### Требования
* Компилятор `GCC` или `Clang` с поддержкой стандарта не ниже `C99`.

### Компиляция
Скомпилируйте исходный код с помощью следующей команды:
```bash
gcc -std=c99 define_variable_macro.c -o define_variable_macro
```

### Примеры использования

#### 1. Успешный запуск (Валидные параметры):
```bash
./define_variable_macro -n "my_data"
```
*Результат:* Программа успешно пройдет все проверки и вернет код `0`.

#### 2. Ошибка: Недостаточно аргументов:
```bash
./define_variable_macro -n
```
*Вывод в stderr:*
```text
Invalid number of arguments for version 2.3.4.
```

#### 3. Ошибка: Неверный управляющий флаг:
```bash
./define_variable_macro -x "my_data"
```
*Вывод в stderr:*
```text
-x is a wrong param at index 1 for version 2.3.4.
```
